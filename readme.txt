Aplicativo Web para gerenciar as reservas de uma pousada


Este é o repositório do aplicativo Gerenciador de Reservas

O serviço de back-end é uma api REST composta de diversos end-points,
cada um responsável por uma determinada entidade do domínio da aplicação.

O serviço de front-end é uma SPA que permite acesso aos serviços de back-end 
mediante autenticação de usuário e autorização de acesso aos endpoint com uso de JWT.

A idéia é validar um MVP que mostre a viabilidade do sistema.

Stack:
Back-end
 * IDE Eclipse
 * Maven
 * Java 17 + Spring Boot 3.1.5
 * flyway para migration
 * MySql Workbench 8.0
 * JWT
 * Postman
Front-end
 * Node.js 20.9.0
 * Angular 16.2.9
 * Bootstrap
 * TypeScript
 * Html / Css
